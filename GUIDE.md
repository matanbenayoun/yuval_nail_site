
## 1. דף הבית (`/`)

### Hero Section
- כותרת ראשית עם קריאה לפעולה
- שני כפתורים: **"הזמיני תור"** (ניווט ל-`/booking`) + **"לגלריה"** (גלילה לקטע הגלריה)
- 3 נתוני אמון: לקוחות מרוצות / שנות ניסיון / מוצרי ג'ל

### קטע שירותים (`#services`)
| שירות | מחיר | משך |
|---|---|---|
| מניקור ג'ל | ₪180 | 60 דק' |
| תוספות ותחזוקה | ₪140 | 45 דק' |
| אמנות ציפורניים | מ-₪220 | 90 דק' |

לחיצה על "הזמיני תור" בתחתית הקטע מנווטת ל-`/booking`.

### גלריה (`#gallery`)
- מציגה את כל התמונות שיובל העלתה דרך לוח הבקרה
- בהתחלה: 6 דוגמיות צבע (gradient) כברירת מחדל
- ריחוף על תמונה מציג את שם הצבע/עיצוב
- כפתור מובייל: שם מוצג תמיד בתחתית התמונה

## 2. הזמנת תור (`/booking`)

תהליך הזמנה בשלושה שלבים רצופים:

### שלב 1 — בחירת תאריך
- לוח שנה אינטראקטיבי
- **ימי ראשון חסומים** (יום מנוחה)
- **תאריכים שעברו חסומים** (לא ניתן לבחור עבר)

### שלב 2 — בחירת שעה
- מופיע **לאחר** בחירת תאריך
- 4 סלוטים: **10:00 / 12:00 / 14:00 / 16:00**
- שעה תפוסה → מוצגת עם קו חוצה + disabled (לא לוחצת)
- שעה תפוסה = כבר קיימת הזמנה באותו תאריך+שעה עם סטטוס שאינו "בוטל"

### שלב 3 — פרטים אישיים
- מופיע **לאחר** בחירת שעה
- שדות: **שירות** (dropdown) + **שם מלא** + **טלפון**
- כפתור "אישור הזמנה" → שולח לכל הסטטים

### אישור הזמנה
- הודעת הצלחה עם פרטי ההזמנה
- כפתורי "הזמנה נוספת" / "לדף הבית"
- ההזמנה נכנסת מיידית ללוח הבקרה עם סטטוס **"ממתין"**

---

## 3. כניסה לניהול (`/admin/login`)

### פרטי התחברות
```
שם משתמש:  yuvalsinreuven
סיסמה:     yuval02092004
```

> לשינוי הפרטים: ערכי קובץ `.env.local` בשורש הפרויקט (ראה סעיף [הגדרות](#הגדרות))

### מה קורה בכניסה
1. הלקוח שולח POST ל-`/api/admin/login`
2. השרת בודק מול משתני הסביבה
3. בהצלחה: מגדיר **cookie מוצפן** (`httpOnly`) + מנתב ל-`/admin`
4. כל ניסיון גישה ל-`/admin/*` ללא cookie → redirect אוטומטי לדף הכניסה

## 4. לוח הבקרה (`/admin`)

### כותרת עליונה
- **כפתור בית** (🏠) → חזרה לאתר הציבורי
- **כפתור יציאה** (→) → logout + redirect לדף הכניסה

### כרטיסיות סטטיסטיקה
| כרטיס | מה מציג |
|---|---|
| **סה"כ** | כל ההזמנות במערכת |
| **היום** | הזמנות שתאריכן הוא היום |
| **ממתינים** | הזמנות עם סטטוס "ממתין" |

---

### טאב תורים 📅

#### חיפוש
שדה חיפוש חופשי — מסנן לפי:
- שם לקוחה
- שם שירות
- מספר טלפון
- תאריך (פורמט `YYYY-MM-DD`)

#### פילטר סטטוס
כפתורי פילטר: **הכל / ממתין / אושר / בוטל** + מספר ב-() לכל קטגוריה

#### כרטיסיית תור
כל תור מציג:
- **תאריך + שעה** (בלוק צבעוני בצד)
- **שם לקוחה** + **טלפון** (לחיצה על הטלפון מתקשרת ישירות!)
- **שם השירות**
- **badge סטטוס** — לחיצה עליו מחזירה את הסטטוס הבא:

```
ממתין → אושר → בוטל → ממתין → ...
```

##### צבעי סטטוס
| סטטוס | צבע |
|---|---|
| ממתין | ענבר (כתום) |
| אושר | ירוק |
| בוטל | אדום |

---

### טאב גלריה 🖼️

#### העלאת תמונות
- לחיצה על **"העלאת תמונה"** → פותח את בורר הקבצים
- ניתן לבחור **מספר תמונות בבת אחת**
- תומך בכל פורמטי תמונה (`image/*`)
- התמונות מופיעות **מיידית** גם בגלריה הציבורית

#### ניהול תמונות
- כל תמונה מוצגת בגריד 2 עמודות
- **ריחוף** מציג כפתור מחיקה (🗑️) בפינה
- לחיצה על המחיקה → מסירה מהגלריה הציבורית מיידית
- שם התמונה מוצג בתחתית כל תמונה

> **שים לב:** הגלריה מאוחסנת זיכרון (state) בלבד — לא בבסיס נתונים. רענון הדף מאפס לתמונות ברירת המחדל. לשמירה קבועה יש צורך בחיבור לשירות אחסון (S3, Cloudinary, וכו').

## 5. הגדרות

### קובץ `.env.local`
```env
ADMIN_USERNAME=yuvalsinreuven      # שם המשתמש לכניסה
ADMIN_PASSWORD=yuval02092004       # הסיסמה לכניסה
ADMIN_SESSION_SECRET=luna-admin-secret-2024   # מחרוזת סודית לחתימת ה-cookie
```

**לשינוי הסיסמה:**
1. ערכי את `.env.local`
2. הפעילי מחדש `npm run dev`

**בפרודקשן:** שיני את `ADMIN_SESSION_SECRET` למחרוזת ארוכה ואקראית.
